# CryptoFlow

A real-time cryptocurrency dashboard built with SolidJS and Binance API.

![SolidJS](https://img.shields.io/badge/SolidJS-2c4f7c?style=flat&logo=solid&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-007ACC?style=flat&logo=typescript&logoColor=white)

## Features

- **Live prices** via WebSocket
- **Top 100 coins** sorted by volume
- **Search & filter** by name or symbol
- **Watchlist** - star your favorites
- **Price charts** - 24H to 1Y views
- **Order book** - live bid/ask levels
- **Recent trades** - real-time feed

## Quick Start

```bash
# Install dependencies
bun install

# Start dev server
bun dev

# Build for production
bun run build
```

## Tech Stack

- [SolidJS](https://solidjs.com) - Reactive UI
- [TypeScript](https://typescriptlang.org) - Type safety
- [Vite](https://vitejs.dev) - Build tool
- [TailwindCSS](https://tailwindcss.com) - Styling
- [Binance API](https://binance-docs.github.io/apidocs/) - Market data

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `/` | Search |
| `W` | Toggle watchlist |
| `R` | Refresh |
| `Esc` | Clear search |
